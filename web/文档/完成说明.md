## 🚀 如何使用

### 方式一：快速启动（推荐新手）

1. **Windows 用户**
   ```
   双击 start.bat
   ```

2. **Mac/Linux 用户**
   ```bash
   chmod +x start.sh
   ./start.sh
   ```

3. **打开浏览器**
   ```
   http://localhost:5000
   ```

### 方式二：手动启动

```bash
# 1. 安装依赖
pip install flask flask-cors

# 2. 配置账号
cp ../config.json.template ../config.json
# 编辑 config.json 添加 SessionID

# 3. 启动服务器
python server.py

# 4. 打开浏览器
# http://localhost:5000
```

### 方式三：手机访问

```bash
# 1. 启动服务器（同上）

# 2. 获取电脑 IP
ipconfig  # Windows
ifconfig  # Mac/Linux

# 3. 手机浏览器打开
http://[你的IP]:5000
```
